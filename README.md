## 📌 Project Overview
This repository contains a comprehensive tutorial and technical demonstration of **Support Vector Machine (SVM) Kernels**. The project illustrates how to handle non-linearly separable data by transitioning from Linear classifiers to Polynomial and Radial Basis Function (RBF) Kernels.

The objective of this tutorial is to educate students and practitioners on the "Kernel Trick" and how to optimize hyperparameters to solve complex classification tasks.

## 🛠️ Key Features
* **Technical Deep Dive:** Comparison of Linear, Polynomial, and RBF Kernels.
* **Hyperparameter Tuning:** Automated optimization using `GridSearchCV`.
* **Visual Learning:** High-contrast decision boundary plots for intuitive understanding.
* **Accessibility:** Designed with colorblind-friendly palettes and structured for screen readers.

## 📂 Repository Structure
* `svm_tutorial.ipynb`: Main Jupyter Notebook containing all code, mathematical explanations, and figures.
* `cluster_moons.csv`: The primary dataset featuring non-linear "moon" shapes.
* `README.md`: Project documentation (this file).
